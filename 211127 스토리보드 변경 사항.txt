변경  사항

- 간호사 메인: 수정 버튼 -> 삭제 버튼, 사유를 메인에서 바로 수정가능하도록
- 간호사 메인: 예약 -> 진료대기 상태 변화, 방법 어떻게?(메인에 버튼 추가 or 예약 페이지에서 수정)
- 간호사 접수: (스토리 보드 제작 완료)
- 간호사 예약 관리: 좌측 상단에 환자 리스트, 좌측 하단에 해당 환자의 예약 리스트, 우측에 예약 신청 폼
- 간호사 예약 관리: 예약 정보 버튼 삭제, 환자 이름 선택 시 예약 정보 표시
- 의사 메인 & 간호사 메인: 각자의 메인 화면 환자 진료 상태 변경시, 상대방에게 푸시 알람 전달
- 의사 메인 화면 - 병명 검색: 질병 창 paging & scroll 처리
- 의사 메인 화면 - 병명 검색: 
- 관리자 메인: (미정)관리자 추가 기능
- 관리자 의약품 리스트 관리: 우측 정보란의 확인 버튼 왼편에 수정 버튼 추가
- 관리자 의약품 리스트 관리: 의약품 삭제 버튼 자리 이동
- 회원 정보 수정: 버튼 이름 수정(회원 가입 -> 회원 정보 수정)
